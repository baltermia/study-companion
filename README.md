# study-companion

A study companion application with Telegram bot integration.

## Projects

### StudyCompanion.Bot

A Telegram bot client for the StudyCompanion application. See [src/StudyCompanion.Bot/README.md](src/StudyCompanion.Bot/README.md) for more details.

## Getting Started

### Prerequisites

- .NET 10.0 or later
- A Telegram Bot Token (obtain from [@BotFather](https://t.me/botfather))

### Building

```bash
dotnet build
```

### Running the Bot

```bash
cd src/StudyCompanion.Bot
dotnet run
```

Make sure to configure your bot token in `appsettings.json` or as an environment variable before running.
